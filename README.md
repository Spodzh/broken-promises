# Broken Promises

Интерактивная визуальная новелла о жизни Марка, родившегося в 2000 году в депрессивном российском городе Заветченск. Игра рассказывает историю от рождения до смерти в 30 лет, где каждый выбор влияет на финал.

## Особенности
- 32-битный пиксельный стиль
- Нелинейный сюжет с 3 концовками
- Сохранение прогресса в браузере
- Около 3 часов игры

## Управление
Читайте текст и нажимайте на кнопки выбора. Прогресс сохраняется автоматически.

## Разработка
Проект создан на HTML, CSS и JavaScript. Для запуска достаточно открыть `index.html` в браузере.

## Лицензия
Свободное использование.
