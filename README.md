# Broken Promises

Интерактивная визуальная новелла о жизни Марка (2000–2030). Каждый выбор влияет на судьбу.

**Версия:** 0.1.3

## Особенности
- 32-битный пиксельный стиль
- 60+ сцен
- 3 концовки (хорошая, средняя, плохая)
- Статистика в конце игры
- Сохранение прогресса
- Адаптация под ПК и мобильные

## Управление
- Читайте текст, нажимайте на кнопки выбора.
- Клик по тексту — ускорить печать.

## Запуск
Откройте `index.html` в браузере.
